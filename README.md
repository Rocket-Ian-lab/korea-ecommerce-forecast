# 📦 한국 전자상거래 수출입 수요예측 시스템
**Korea E-Commerce Trade Demand Forecasting System**

> 관세청 전자상거래 무역통계를 기반으로 SARIMAX 시계열 모델과 한국 이커머스 이벤트 회귀를 결합하여  
> 2026년 월별 수출입 금액을 예측하고, Streamlit 대시보드 및 PDF 보고서로 시각화하는 AI 분석 플랫폼입니다.

---

## 🗂️ 프로젝트 구조

```
수요예측모델/
│
├── ecommerce_forecast.py        # 핵심 분석 스크립트 (전처리 → EDA → 모델 → 예측 → 저장)
├── generate_report.py           # PDF 보고서 자동 생성 스크립트
├── app.py                       # Streamlit 인터랙티브 대시보드
│
├── eda_trend.png                # EDA 추세 차트 (수출입 + YoY + 건수)
├── components_export.png        # 수출 시계열 분해 (추세/계절성/잔차)
├── components_import.png        # 수입 시계열 분해 (추세/계절성/잔차)
├── forecast_result.png          # 예측 결과 차트 (실적 + 예측 + 신뢰구간)
├── forecast_summary_bar.png     # 2026년 월별 예측 바 차트
│
├── 2026_ecommerce_forecast.csv  # 예측 결과 데이터 (UTF-8 BOM)
└── 2026_ecommerce_forecast_report.pdf  # 최종 PDF 보고서 (7페이지)
```

## 🚀 실행 방법

### 1️⃣ 의존성 설치

```bash
pip install pandas numpy matplotlib statsmodels \
            streamlit plotly reportlab pillow \
            openpyxl xlsxwriter
```

### 2️⃣ 핵심 분석 실행 (차트 + CSV 생성)

```bash
python -X utf8 ecommerce_forecast.py
```

**생성 파일:**
- `eda_trend.png` — EDA 추세 차트
- `components_export.png` / `components_import.png` — 시계열 분해
- `forecast_result.png` — 예측 결과 차트
- `forecast_summary_bar.png` — 바 차트
- `2026_ecommerce_forecast.csv` — 예측 결과 CSV

### 3️⃣ PDF 보고서 생성

```bash
python -X utf8 generate_report.py
```

**생성 파일:**
- `2026_ecommerce_forecast_report.pdf` (A4, 7페이지, 한글 지원)

### 4️⃣ Streamlit 대시보드 실행

```bash
python -m streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

## ⚠️ 환경 참고사항

> **Python 3.14 환경**에서는 Facebook Prophet의 Stan 백엔드 빌드 도구(mingw32-make)가 없어  
> Prophet 초기화가 불가합니다. 이를 대체하여 **statsmodels SARIMAX**로 동일한 분석을 구현하였습니다.  
> Prophet과 동일하게 추세 + 계절성 + 이벤트 효과를 분리하여 예측합니다.
